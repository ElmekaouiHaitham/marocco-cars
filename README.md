# 🚗 Cars Mar — Autonomous AI Used Car Scouting Agent

**Cars Mar** is a cutting-edge autonomous AI agent designed to hunt, evaluate, and negotiate the best used car deals on the Moroccan market (specifically Avito.ma). 

Unlike traditional web scrapers that simply dump data, Cars Mar utilizes a **Reasoning + Acting (ReAct)** loop powered by Google Gemini. The agent physically controls a browser, scrolls through listings, reads descriptions, calculates market baselines, and flags suspicious deals based on human-like intuition.

## ✨ Features

- **🧠 Fully Autonomous Scouting:** Give the agent a target model and budget, and it will browse the web to find it.
- **⚡ Fast Baseline Pre-computation:** The system quickly scans hundreds of preview listings via HTTP requests to calculate a live market baseline (median price, expected price per year, average mileage) for accurate deal evaluation.
- **🕵️ Deep Evaluation & Red Flags:** The AI reads through descriptions to detect hidden accident reports, missing photos, urgent sales, or suspicious "dédouanement" (customs) issues.
- **💬 Negotiation Tips:** For every approved listing, the AI generates a customized, 1-sentence negotiation strategy based on the seller's urgency and market data.
- **📊 Real-time Dashboard:** A sleek, dark-themed Next.js dashboard visualizes the findings, ranking them by score and providing clear trust signals.

## 🏗️ Architecture

The project is split into two main components:

### 1. The Autonomous Agent (`/backend`)
Built in Python, the backend leverages:
- **Selenium:** Controls a visible Google Chrome window, allowing the agent to navigate, scroll, and click.
- **Google Gemini API:** The brain of the operation, using function calling to decide when to click links, go back, or evaluate a deal.
- **BeautifulSoup & Requests:** Used strictly for the fast, initial pre-computation of the market baseline to keep API costs down.

### 2. The Dashboard (`/frontend`)
Built with Next.js 14 (App Router) and Tailwind CSS, the frontend:
- Reads the dynamically updating `results.json` generated by the agent.
- Displays evaluated cars in a glassmorphism-styled grid.
- Highlights "Great Deals" vs "Average" cars with visual badges.

## 🚀 Getting Started

### Prerequisites
- Python 3.11+
- Node.js 18+
- Google Gemini API Key

### Backend Setup (The AI Agent)

1. Navigate to the backend directory:
   ```bash
   cd backend
   ```
2. Create and activate a virtual environment:
   ```bash
   python -m venv venv
   # On Windows:
   .\venv\Scripts\activate
   ```
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Create a `.env` file and add your Gemini API Key:
   ```env
   GEMINI_API_KEY=your_api_key_here
   ```
5. Run the Agent:
   ```bash
   python main.py
   ```
   *You will see the terminal print the agent's internal thoughts while the Chrome browser autonomously navigates the listings!*

### Frontend Setup (The Dashboard)

1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) to view the live dashboard.

## 📝 Configuration

You can easily change what the agent looks for by modifying `backend/config.py`:
```python
SEARCH_CONFIG = {
    "model": "Dacia Logan",
    "cities": ["Casablanca", "Rabat", "Salé", "Marrakech"],
    "min_year": 2022
}
```

## ⚠️ Important Note on Rate Limits
This project uses the Gemini API. If you are on the **Free Tier**, you may encounter the `5 Requests Per Minute (RPM)` limit, as the autonomous ReAct loop fires multiple reasoning queries per minute (Read Page -> Think -> Click -> Read -> Evaluate). 

For extended autonomous runs without manual delays, a paid API tier is recommended.
